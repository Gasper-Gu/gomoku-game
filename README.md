# Gomoku Game - 五子棋

一款基于 HTML5 Canvas 和原生 JavaScript 实现的五子棋游戏，内置 AI 对手，支持多种难度选择。

## 功能特性

- 🎮 **完整五子棋规则**：15x15 标准棋盘，黑棋先手，五子连珠获胜
- 🤖 **智能 AI 对手**：三种难度（业余/中等/高手），基于启发式评分算法
- 🎨 **现代 UI 设计**：暗黑主题 + 渐变色彩 + 响应式布局
- 🖱️ **交互体验**：
  - 点击落子，自动标记最后一步
  - 悔棋功能（支持撤销两步）
  - 游戏结果浮层提示
- 🎭 **视觉特效**：
  - 棋子渐变阴影效果
  - 星位（天元）标注
  - 活动玩家指示器
  - 结局动画效果

## 难度说明

| 难度 | 说明 | AI 行为 |
|------|------|---------|
| 业余 (Amateur) | 适合新手 | 只考虑前 3 个高分位置，随机选择 |
| 中等 (Medium) | 适合普通玩家 | 考虑前 5 个位置，加权随机，注重防守 |
| 高手 (Expert) | 挑战资深玩家 | 考虑前 10 个位置，加权随机，防守优先 |

## 使用方法

直接在浏览器中打开 `index.html` 即可开始游戏。

### 操作说明

- **落子**：点击棋盘交叉点
- **重新开始**：点击 "New Game" 按钮
- **悔棋**：点击 "Undo" 按钮（可撤销两步）
- **切换难度**：在下拉菜单中选择 AI 难度

## 技术实现

- **Canvas API**：用于绘制棋盘和棋子
- **启发式算法**：AI 通过评估棋型（连五、活四、冲四、活三等）和位置权重进行决策
- **响应式设计**：自动适配不同屏幕尺寸

## AI 算法简介

AI 使用评分系统评估每个空位的价值：

```
评分 = 进攻分数 + 防守分数 × 防守权重
```

评估维度包括：
- 连珠数量（连五 > 活四 > 冲四 > 活三 > 眠三 > 活二 > 眠二）
- 开放端数（活点 > 眠点）
- 位置权重（靠近中心的棋更值钱）
